## **模块二十六：CSS 盒模型 (续)**

### **1. CSS 内边距 (Padding)**

- **定义**: CSS 的 `padding` 属性用于在内容周围生成空间。`padding` 属性设置的是元素内容与元素边框之间的空白区域大小。

- **单独边距属性**:

  CSS 具有为元素每一侧指定内边距的属性：

  - `padding-top`
  - `padding-right`
  - `padding-bottom`
  - `padding-left`

  所有 `padding` 属性都可以有以下值：

  - `length` - 以 px, pt, cm 等单位指定内边距。
  - `%` - 以包含块宽度的百分比指定内边距。
  - `inherit` - 指定内边距应从父元素继承。

- **简写属性 (`padding`)**:

  `padding` 属性是 `padding-top`, `padding-right`, `padding-bottom`, 和 `padding-left` 的简写属性。

### **2. CSS 尺寸 (Dimensions)**

- **定义**: CSS 尺寸属性允许你控制一个元素的高度和宽度。
- **`height` 和 `width` 属性**:
  - 可以设置为 `auto`（默认值，浏览器自动计算高宽），或以长度单位（px, cm 等）或包含块的百分比（%）来指定。
  - `height` 和 `width` 属性不包括内边距、边框或外边距；它们设置的是元素内边距、边框和外边距**内部**区域的高度/宽度。
- **`max-width` 属性**:
  - 用于设置一个元素的最大宽度。
  - 当浏览器窗口小于元素的 `width` 时，浏览器会添加一个水平滚动条。在这种情况下，使用 `max-width` 将改善浏览器对小窗口的处理。
  - `max-width` 的值会覆盖 `width` 的值。

## **模块二十七：CSS 文本与字体**

### **1. CSS 文本格式化**

- **文本对齐 (`text-align`)**:
  - 该属性用于设置文本的水平对齐方式。文本可以左对齐、右对齐、居中或两端对齐。
  - 当 `text-align` 设置为 "justify" 时，每行都会被拉伸以使每行具有相等的宽度，并且左右边距相等。
- **文本装饰 (`text-decoration`)**:
  - 用于设置或移除文本的装饰。
  - `text-decoration: none;` 通常用于移除链接的下划线。
  - 其他值可用于装饰文本，如 `overline`（上划线），`line-through`（删除线），和 `underline`（下划线）。
- **文本转换 (`text-transform`)**:
  - 用于指定文本中的大写和小写字母。
  - 它可以将所有内容转换为大写或小写，或者将每个单词的首字母大写。
- **文本缩进 (`text-indent`)**:
  - 用于指定文本第一行的缩进。

### **2. CSS 字体 (Fonts)**

- **定义**: CSS 字体属性定义了文本的字体族、粗细、大小和样式。

- **字体族 (`font-family`)**:

  在 CSS 中，有两种字体族名称：

  - **通用字体族 (generic family)** - 一组外观相似的字体族（如 "Serif" 或 "Monospace"）。
  - **特定字体族 (font family)** - 一个特定的字体族（如 "Times New Roman" 或 "Arial"）。
  - `font-family` 属性用于设置文本的字体族。
  - 该属性应包含多个字体名称作为一个“后备”系统。如果浏览器不支持第一个字体，它会尝试下一个，以此类推。应以你想要的字体开始，并以一个通用字体族结束。
  - **注意**: 如果字体族名称包含多个单词，必须用引号括起来，如 `"Times New Roman"`。

- **字体样式 (`font-style`)**:

  - `font-style` 属性主要用于指定斜体文本。
  - `normal` - 文本正常显示。
  - `italic` - 文本以斜体显示。
  - `oblique` - 文本“倾斜”（oblique 与 italic 非常相似，但支持性较差）。

- **字体大小 (`font-size`)**:

  - `font-size` 属性设置文本的大小。
  - 你不应该使用字体大小调整来使段落看起来像标题，或标题看起来像段落。应始终使用正确的 HTML 标签，如 `<h1>` - `<h6>` 用于标题，`<p>` 用于段落。
  - **绝对尺寸 (Absolute size)**: 将文本设置为指定大小。不允许用户在所有浏览器中更改文本大小（不利于可访问性）。
  - **相对尺寸 (Relative size)**: 相对于周围元素设置大小。允许用户在浏览器中更改文本大小。
  - **使用像素 (px)**: 使用像素设置文本大小可以让你完全控制文本大小。
  - **使用 Em (`em`)**: 许多开发者使用 `em` 代替像素，以允许用户调整文本大小。`em` 尺寸单位是 W3C 推荐的。1em 等于当前的字体大小。浏览器中的默认文本大小是 16px，所以 1em 的默认大小是 16px。尺寸转换公式：`pixels/16=em`。

## **模块二十八：CSS 链接与列表**

### **1. CSS 链接 (Links)**

- **定义**: 使用 CSS，链接可以被以不同方式设置样式。链接可以用任何 CSS 属性进行样式化（例如 `color`, `font-family`, `background` 等）。

- **链接状态**:

  链接可以根据其所处的状态进行不同的样式化。四个链接状态是：

  - `a:link` - 一个正常的、未访问的链接。
  - `a:visited` - 用户已访问过的链接。
  - `a:hover` - 用户鼠标悬停在其上时的链接。
  - `a:active` - 链接被点击的瞬间。
  - **注意**: 为了生效，`a:hover` 必须在 CSS 定义中位于 `a:link` 和 `a:visited` 之后。`a:active` 必须在 CSS 定义中位于 `a:hover` 之后。

- **链接样式设置**:

  - `text-decoration` 属性主要用于移除链接的下划线。
  - `background-color` 属性可以用来为链接指定背景颜色。

- **高级：链接按钮**:

  - 通过结合多个 CSS 属性，可以将链接显示为方框/按钮。
  - `display: inline-block` 允许你在元素上设置 `width` 和 `height`，这与 `display: inline` 不同。

------

### **2. CSS 列表 (Lists)**

- **定义**: 在 HTML 中，主要有两种类型的列表：无序列表 (`<ul>`) 和有序列表 (`<ol>`)。CSS 列表属性允许你为列表设置不同的项目标记、使用图像作为标记，以及添加背景颜色。
- **列表项标记**:
  - `list-style-type` 属性指定列表项标记的类型。
  - `list-style-image` 属性指定一个图像作为列表项标记。
- **标记位置 (`list-style-position`)**:
  - 指定列表项标记应出现在内容流的内部还是外部。
  - `inside`: 列表项标记被放置在文本内部，换行文本会根据标记对齐。
  - `outside`: (默认设置) 列表项标记被放置在文本外部，换行文本不会根据标记对齐。
- **简写属性 (`list-style`)**:
  - `list-style` 是一个简写属性，用于在一个声明中设置所有列表属性。
  - 当使用简写属性时，属性值的顺序是：`list-style-type`, `list-style-position`, `list-style-image`。
- **为列表添加颜色**:
  - 添加到 `<ol>` 或 `<ul>` 标签的任何样式会影响整个列表，而添加到 `<li>` 标签的属性会影响单个列表项。

## **模块二十九：CSS 表格 (Tables)**

- **定义**: HTML 表格的外观可以通过 CSS 大大改善。
- **表格边框**:
  - 要在 CSS 中指定表格边框，使用 `border` 属性。
  - 对 `<table>`, `<th>`, 和 `<td>` 元素同时设置边框会导致双重边框，因为它们各自都有独立的边框。
- **折叠边框 (`border-collapse`)**:
  - `border-collapse` 属性设置表格边框是否应合并为单个边框。
  - 如果你只想在表格周围有一个边框，只需为 `<table>` 指定 `border` 属性。
- **表格尺寸 (`width` 和 `height`)**:
  - 表格的宽度和高度由 `width` 和 `height` 属性定义。
- **对齐**:
  - **水平对齐 (`text-align`)**: 设置 `<th>` 或 `<td>` 中内容的水平对齐方式。默认情况下，`<th>` 元素内容居中对齐，`<td>` 元素内容左对齐。
  - **垂直对齐 (`vertical-align`)**: 设置 `<th>` 或 `<td>` 中内容的垂直对齐方式（如 `top`, `bottom`, `middle`）。默认情况下，垂直对齐方式是 `middle`。
- **表格样式技巧**:
  - **水平分隔线**: 为 `<th>` 和 `<td>` 添加 `border-bottom` 属性以创建水平分隔线。
  - **悬停高亮**: 在 `<tr>` 上使用 `:hover` 选择器可以在鼠标悬停时高亮表格行。
  - **斑马条纹表格**: 使用 `nth-child(even)` 选择器为所有偶数（或奇数）表格行添加背景色，以创建斑马条纹效果。
  - **表头颜色**: 可以为 `<th>` 元素指定背景颜色和文本颜色来美化表头
