### **模块二：HTML 基础**

#### **2. HTML 文档基本结构**

一个基础的 HTML 文档包含以下几个部分：

- `<!DOCTYPE html>`：定义文档类型为 HTML。所有 HTML 文档都必须以此声明开头。
- `<html>...</html>`：描述整个 HTML 文档的根元素。
- `<head>...</head>`：提供关于文档的元数据（meta data），如标题、样式表等，这部分内容不会在页面上显示。
- `<title>...</title>`：提供文档的标题，通常显示在浏览器的标签页上。
- `<body>...</body>`：包含页面的所有可见内容，如标题、段落、图片等。这部分内容是唯一会被浏览器直接显示出来的区域。

**结构示例：**



```
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

</body>
</html>
```





#### **3. HTML 标签 (Tags)**

- **定义**：HTML 标签是由尖括号包围的关键词，例如 `<tagname>content</tagname>`。

- 配对

  ：标签通常成对出现，如 

  ```
  <p>
  ```

   和 

  ```
  </p>
  ```

  。

  - 第一个标签是**开始标签**，也称为开放标签。
  - 第二个标签是**结束标签**，在标签名前有一个斜杠。

- **大小写**：HTML 标签不区分大小写（如 `<P>` 和 `<p>` 含义相同），但推荐在 HTML4 或更严格文档中使用小写。

#### **4. HTML 元素 (Elements)**

- **定义**：HTML 元素指从开始标签到结束标签的全部内容。例如：`<p>My first HTML paragraph.</p>`。

- 嵌套元素

  ：HTML 元素可以相互嵌套。例如：

  - `<body>` 元素嵌套在 `<html>` 中。
  - `<h1>` 和 `<p>` 元素又嵌套在 `<body>` 中。

- **空元素**：没有内容的 HTML 元素称为空元素，如 `<br>`。在 HTML5 中不需要关闭，但可以写为 `<br />`。

- **结束标签的重要性**：某些标签即使省略结束标签也能显示，但不推荐依赖此特性，以免导致错误。

### **模块四：HTML 核心概念**

#### **1. HTML 属性 (Attributes)**

- **定义**：属性为 HTML 元素提供额外信息，规定在**开始标签**中。
- **格式**：`名称="值"`。
- **引号**：推荐始终使用引号，尤其是双引号。属性值包含双引号时可使用单引号。
- **大小写**：HTML5 对属性名称不强制小写，但建议统一使用小写。

**常见属性举例：**

- `lang`: 声明文档语言，例如 `<html lang="en-US">`。
- `title`: 显示为鼠标悬停时的提示。
- `href`: 用于 `<a>` 标签定义链接目标，例如 `<a href="http://example.com">Link</a>`。

#### **2. 标题 (Headings)**

- `<h1>` 到 `<h6>` 定义标题，`<h1>` 最重要，`<h6>` 最次。
- 搜索引擎会使用标题结构来索引页面内容。
- 浏览器会自动在标题前后添加外边距。

#### **3. 段落与分隔**

- **段落 (`<p>`)**：定义一个段落，浏览器自动添加空白。

- **换行 (`<br>`)**：不换段落但换行，用于内容中断。

- **水平线 (`<hr>`)**：用于分隔内容。

- **显示规则**：HTML 中的多个空格或换行只会显示一个空格。

#### **4. 文本格式化**(考到不会)

**HTML 使用特定元素进行文本强调和格式设置：**

- **`<b>`: 粗体，无语义强调。**
- **`<strong>`: 强调，具有语义意义。**
- **`<i>`: 斜体，无语义强调。**
- **`<em>`: 强调，具有语义意义。**
- **`<mark>`: 高亮文本。**
- **`<small>`: 小号字体。**
- **`<sub>`: 下标。**
- **`<sup>`: 上标。**
- **`<del>`: 删除线。**
- **`<ins>`: 插入/下划线。**

提示：浏览器常将 `<strong>` 和 `<b>` 显示为粗体，但 `<strong>` 有语义意义；同理 `<em>` 和 `<i>`。

- 使用 `style` 属性设置内联样式。
- 语法：`style="属性:值;"`。

**常用样式属性包括：**

- `background-color`: 背景颜色。

#### **6. HTML 注释 (Comments)**

- 语法：`<!-- 注释内容 -->`。
- 浏览器不会显示注释内容。
- 可用于说明、调试、临时隐藏代码等。



```
<!-- 这是一个注释 -->
<p>This is a paragraph.</p>
```

### **模块五：HTML 文档头部与元数据**

#### **1. `<!DOCTYPE>` 声明**

- 帮助浏览器正确解析网页。
- 不区分大小写。
- HTML5 的标准声明为：`<!DOCTYPE html>`。

#### **2. `<head>` 元素**

- `<head>` 是所有元数据的容器，位于 `<html>` 与 `<body>` 之间。

常见元标签：

- `<title>`：定义网页标题。
- `<meta>`：设置字符编码等元信息，如 `<meta charset="UTF-8">`。
- `<style>`：内部 CSS。
- `<link>`：外部 CSS 链接。
