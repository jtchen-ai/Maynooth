## **模块三十四：CSS 浮动与清除**

### **1. `float` 属性**

- `float` 属性指定一个元素是否应该浮动。

- 它可以用来实现图像环绕文本的效果。

- 属性值

  :

  - `none`: (默认值) 元素不浮动，将显示在它在文本中出现的位置。
  - `left`: 元素向左浮动。
  - `right`: 元素向右浮动。
  - `initial`: 将此属性设置为其默认值。
  - `inherit`: 从其父元素继承此属性。

------

### **2. `clear` 属性**

- `clear` 属性用于控制浮动元素的行为。

- 浮动元素之后的元素会环绕它。为避免这种情况，可以使用 `clear` 属性。

- `clear` 属性指定了在一个元素的哪一侧不允许有浮动元素。

- 属性值

  :

  - `none`: (默认值) 允许两侧都有浮动元素。
  - `left`: 左侧不允许有浮动元素。
  - `right`: 右侧不允许有浮动元素。
  - `both`: 左侧或右侧都不允许有浮动元素。

## **模块三十五：CSS 导航栏**



### **1. 导航栏基础**

- 导航栏需要标准的 HTML 作为基础。

- 导航栏本质上是一个链接列表，因此使用 `<ul>` 和 `<li>` 元素是完全合理的。

- 标准初始代码

  :

  - `list-style-type: none;` - 移除项目符号，因为导航栏不需要列表标记。
  - `margin: 0;` 和 `padding: 0;` - 移除浏览器的默认设置。
  - 这段代码是垂直和水平导航栏通用的标准代码。

------

### **2. 垂直导航栏**

- **基本实现**:
  - 将链接 `<a>` 显示为块级元素 (`display: block;`)，这使得整个链接区域都可点击，并允许我们指定宽度。
  - 因为块级元素默认会占据全部可用宽度，所以需要为 `<a>` 或 `<ul>` 指定一个宽度。
- **样式技巧**:
  - **当前链接**: 添加一个 "active" 类到当前链接，以告知用户他所在的页面。
  - **居中与边框**: 使用 `text-align: center` 来居中链接。为 `<ul>` 添加 `border` 属性可以在导航栏周围添加边框。为除最后一个 `<li>` 元素外的所有 `<li>` 元素添加 `border-bottom` 可以在内部创建分隔线。
  - **固定导航栏**: 创建一个全屏高的“粘性”侧边导航栏。

------

### **3. 水平导航栏**

有两种方法可以创建水平导航栏：使用内联列表项或浮动列表项。

- **内联列表项 (`display: inline`)**:
  - 将 `<li>` 元素指定为内联 (`display: inline;`)，可以移除列表项前后的换行符，使它们显示在同一行。
- **浮动列表项 (`float: left`)**:
  - 另一种方法是浮动 `<li>` 元素。
  - 使用 `float: left;` 让块级元素并排排列。
  - 将链接 `<a>` 设置为 `display: block;`，使整个链接区域可点击并允许指定内边距。
  - 为 `<ul>` 添加 `overflow: hidden` 可以防止 `<li>` 元素超出列表范围。
- **样式技巧**:
  - **右对齐链接**: 通过将列表项向右浮动 (`float: right;`) 来实现链接的右对齐。
  - **边框分隔符**: 为 `<li>` 添加 `border-right` 属性来创建链接分隔符。
  - **固定导航栏**: 使导航栏固定在页面的顶部或底部，即使用户滚动页面也不移动。使用 `position: fixed;` 来实现。

## **模块三十六：CSS 表单样式**

HTML 表单的外观可以通过 CSS 大大改善。

- **样式化输入框**:
  - 使用 `width` 属性确定输入字段的宽度。
  - 使用属性选择器可以只对特定类型的输入进行样式化，例如 `input[type=text]`。
- **内边距 (`padding`)**:
  - 使用 `padding` 属性在文本字段内部添加空间。
  - **`box-sizing: border-box;`**: 设置此属性可以确保内边距和边框被包含在元素的总宽度和高度之内。
- **边框 (`border`)**:
  - 使用 `border` 属性改变边框大小和颜色，使用 `border-radius` 添加圆角。
  - 如果你只想有一个底部边框，使用 `border-bottom` 属性。
- **颜色**:
  - 使用 `background-color` 属性为输入框添加背景色，使用 `color` 属性改变文本颜色。
- **焦点状态 (`:focus`)**:
  - 默认情况下，一些浏览器在输入框获得焦点（被点击）时会添加一个蓝色轮廓。
  - 你可以通过向输入框添加 `outline: none;` 来移除此行为。
  - 使用 `:focus` 选择器可以在输入字段获得焦点时对其进行样式化。
- **带图标的输入框**:
  - 如果你想在输入框内放置一个图标，可以使用 `background-image` 属性，并通过 `background-position` 属性来定位它。
  - 同时需要添加一个较大的左内边距来为图标预留空间。
- **动画搜索框**:
  - 使用 CSS3 `transition` 属性可以在搜索输入框获得焦点时为其宽度添加动画效果。
  - CSS3 过渡效果允许你在给定的时间内平滑地改变属性值。
- **样式化其他表单元素**:
  - **Textareas**: 使用 `resize` 属性来防止文本区域被调整大小（禁用右下角的“抓手”）。
  - **Select Menus**: 可以为下拉菜单设置样式。
  - **Input Buttons**: 可以为输入按钮设置样式。

## **模块三十七：CSS 分页 (Pagination)**

- **定义**: 如果你的网站有很多页面，你可能希望为每个页面添加某种形式的分页。
- **实现**:
  - 要创建一个分页，可以对一个 HTML 列表进行样式化。
  - **激活与悬停状态**: 用 `.active` 类高亮当前页面，并使用 `:hover` 选择器在鼠标移过页面链接时改变其颜色。
  - **圆角**: 添加 `border-radius` 属性可以得到一个圆角的“激活”和“悬停”按钮。
  - **过渡效果**: 为页面链接添加 `transition` 属性可以在悬停时创建过渡效果。
  - **边框**: 使用 `border` 属性为分页添加边框。
  - **链接间距**: 如果你不想将页面链接组合在一起，可以添加 `margin` 属性。
  - **分页大小**: 使用 `font-size` 属性改变分页的大小。
  - **居中分页**: 要使分页居中，可以用一个容器元素（如 `<div>`）包裹它，并添加 `text-align: center;`。
- **面包屑 (Breadcrumbs)**:
  - 分页的另一种变体是所谓的“面包屑”。
  - `:before` 选择器在所选元素的内容之前插入内容。
