### **思维导图：闪存 (Flash RAM) 和 FPGA**

- #### **闪存 (Flash RAM)**

  - 基本概念

    - 从EEPROM演变而来，可以进行写入和擦除操作 

      。

    - 像RAM一样工作，但在断电时仍能保存内容 

      。

    - 与RAM/ROM的对比

      - 计算机使用RAM进行读写，使用ROM进行存储和读取 

        。

      - RAM（动态或静态）在断电时会清除其内容 

        。

      - 某些ROM可以写入，例如可编程ROM（PROM）、可擦除PROM（EPROM）和电可擦除PROM（EEPROM） 

        。

  - 工作原理：浮栅闪存单元 (Floating gate flash memory cell)

    - 结构

      - 由一个带有控制栅和浮动栅的存储晶体管组成 

        。

      - 浮动栅通过薄氧化层与晶体管的其余部分绝缘 

        。

      - 浮动栅存储电荷并控制电流的流动 

        。

    - 数据写入 (代表 '1')

      - 在高电场作用下，电子被迫穿过薄氧化层到达浮动栅 

        。

      - 这使得电流可以轻易地在沟道中流动，代表二进制值 '1' 

        。

    - 数据擦除 (代表 '0')

      - 通过在控制栅上施加一个强的负电荷来擦除数据 

        。

      - 这将电子推入沟道 

        。

      - 此过程导致电导率大幅下降，代表 '0' 

        。

    - 非易失性

      - 由于氧化层产生的电绝缘性，被困在浮动栅中的电子不依赖于电源 

        。

      - 这意味着闪存可以提供持久性存储 

        。

  - 闪存类型：NAND 和 NOR

    - 它们在架构和设计特性上有所不同 

      。

    - NOR 闪存

      - 采用并行连接的独立存储单元，无共享组件，支持对所有数据的随机访问 

        。

      - 以字节为单位对数据进行编程 

        。

      - 数据读取速度快，但擦除和写入速度通常比NAND慢 

        。

      - 生产成本通常更高 

        。

    - NAND 闪存

      - 单元更紧凑，位线更少，通过串联浮栅晶体管来增加存储密度 

        。

      - 更适合串行数据访问而非随机数据访问 

        。

      - 以页（pages）为单位对数据进行编程，页大于字节但小于块 

        。

      - 对于写密集型应用，功耗低于NOR闪存 

        。

  - 优缺点

    - 优点

      - 是成本最低的半导体存储器形式 

        。

      - 具有非易失性、功耗更低，并可以大块擦除的特点 

        。

      - NOR闪存随机读取速度快，而NAND闪存串行读写速度快 

        。

      - 采用NAND闪存芯片的SSD性能显著高于传统存储介质（如HDD） 

        。

      - 由于没有移动部件，比HDD更可靠 

        。

      - 配备闪存的企业存储系统能够实现以微秒或毫秒为单位的低延迟 

        。

    - 缺点

      - SSD比同等容量的HDD昂贵得多 

        。

      - 目前HDD的容量更大 

        。例如，HDD可达12TB，而最大的SSD容量仅为2TB 

        。

      - 存在“损耗机制”，过多的编程/擦除周期会损坏存储单元 

        。丢失足够多的比特位后，整个驱动器可能无法运行 

        。

      - 从损坏的SSD中恢复数据几乎不可能，而从损坏的HDD中恢复数据则相对容易 

        。

      - 制造商通过内置控制电路来检查潜在的故障，以确保SSD不会在没有警告的情况下耗尽寿命 

        。

- #### **固态硬盘 (Solid State Drive - SSD)**

  - 基本概念

    - 基于硅基板的闪存介质，被称为固态设备 

      。

    - 广泛用于消费电子产品和大型数据存储系统 

      。

  - 形态规格 (Form Factors)

    - 可以安装在传统HDD插槽中，并使用SATA或SCSI等标准HDD接口 

      。

    - 构建在印刷电路板上的固态卡，使用PCIe等标准I/O接口 

      。

    - 安装在DIMM或SO-DIMM插槽中的固态模块，使用SATA等标准HDD接口 

      。

    - 混合硬盘，它结合了传统HDD（用于大容量存储）和NAND闪存模块（用于快速I/O） 

      。

  - 全闪存 SSD

    - 与基于磁盘的存储设备相比，SSD在性能上具有优势，并可能降低运营成本 

      。

    - 其主要区别在于写入数据的底层物理架构 

      。HDD依赖驱动臂，而全闪存存储系统没有移动部件 

      。

    - 能够插入基于x86的个人电脑和服务器中，增加了该技术的普及性 

      。

    - 对于个人用户，SSD成本更高，通常只用于更昂贵的个人电脑 

      。

    - 对于公司而言，在服务器中使用闪存的缺点是需要内部构建硬件系统并从第三方购买存储管理软件 

      。

- #### **现场可编程门阵列 (FPGA)**

  - 基本概念

    - 是可以被配置以实现不同功能的固态设备 

      。

    - 与由程序控制固定电路的微控制器不同，FPGA可以被编程成为不同的数字电路 

      。

    - 可以被重新编程以实现完全不同的功能 

      。

  - 应用与优势

    - 工程师可以使用FPGA并编写硬件描述语言(HDL)来为产品（如洗衣机）进行原型设计 

      。

    - 对于大众消费品，使用FPGA制作原型更便宜，因为HDL程序可用于建立掩模以制造低成本的ASIC芯片 

      。

  - FPGA 结构

    - 

      I/O块 (IOB)

      ：允许FPGA与外部世界连接 

      。

    - 

      软逻辑块

      ：包含查找表、多路复用器和移位寄存器，可用于构建任何组合或时序电路 

      。

    - 

      硬逻辑块

      ：在此图中由内存块和乘法器组成 

      。实际的FPGA可能还包括通信块或整个微处理器 

      。

    - 

      互连资源

      ：开关块、连接块和布线通道允许在软逻辑块内部及不同块之间进行灵活连接 

      。

    - 

      编程机制

      ：HDL程序使用闪存来设置不同部分之间的连接 

      。这些连接可以在需要时被擦除，以用于调试或复用FPGA 

      。

- #### **Verilog**

  - 基本概念

    - 是一种硬件描述语言(HDL)，常用于教学 

      。

    - 与工业界使用的VHDL相比，Verilog更简单易学，并且能完成几乎所有VHDL的功能 

      。
