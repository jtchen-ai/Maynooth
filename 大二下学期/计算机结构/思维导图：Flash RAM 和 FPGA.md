# 思维导图：Flash RAM 和 FPGA

- ## **闪存 (Flash RAM)**

  - **基本概念**

    - 从EEPROM演变而来，可以进行写入和擦除操作 

      。

    - 像RAM一样工作，但在断电时仍能保存内容 

      。

    - 与RAM/ROM的对比

      - RAM（随机存取存储器）：可读可写，但断电后内容会丢失，分为动态和静态两种 

        。

      - ROM（只读存储器）：用于存储和读取，通常不可写入 

        。

      - 特殊ROM：部分ROM可写入，如只能写入一次的PROM，可通过紫外线擦除的EPROM，或通过高电压擦除的EEPROM 

        。

  - **工作原理：浮栅闪存单元 (Floating gate flash memory cell)**

    - 结构

      - 由一个存储晶体管构成，包含一个控制栅和一个浮动栅 

        。

      - 浮动栅通过薄氧化层与晶体管的其他部分绝缘 

        。

      - 浮动栅负责存储电荷并控制电流的流动 

        。

    - 数据写入 (代表二进制 '1')

      - 在强电场作用下（控制栅接强正电，源极和漏极接强负电），电子被迫穿过薄氧化层进入浮动栅 

        。

      - 这使得电流可以轻易在P型掺杂沟道中流动，代表二进制值 '1' 

        。

    - 数据擦除 (代表二进制 '0')

      - 通过在控制栅上施加强负电荷来擦除数据 

        。

      - 这会将电子推入沟道，而沟道中存在强正电荷 

        。

      - 此过程会移除P型掺杂沟道中的大部分空穴，导致电导率大幅下降，代表 '0' 

        。

    - 非易失性

      - 由于氧化层的电绝缘性，被困在浮动栅中的电子不依赖于电源 

        。

      - 因此，闪存可以提供持久性存储，内容可保存多年 

        。

  - **闪存类型：NAND 和 NOR**

    - 两者在架构和设计特性上有所不同 

      。

    - NOR 闪存

      - 架构：无共享组件，存储单元并联，可对所有数据进行独立的随机访问 

        。

      - 读写速度：数据读取速度快，但擦除和写入速度通常比NAND慢 

        。

      - 编程单位：以字节为单位对数据进行编程 

        。

      - 成本：生产成本通常更高 

        。

    - NAND 闪存

      - 架构：更紧凑，浮栅晶体管串联以增加存储密度，位线更少 

        。

      - 访问方式：更适合串行数据访问而非随机访问 

        。

      - 读写速度：擦除和写入速度比NOR快，但读取速度较慢 

        。

      - 编程单位：以页(pages)为单位对数据进行编程，页大于字节但小于块 

        。

      - 功耗：在写入密集型应用中，功耗低于NOR闪存 

        。

  - **优缺点**

    - 优点

      - 成本：最便宜的半导体存储器形式 

        。

      - 特性：非易失性（与DRAM和SRAM不同），功耗更低，可以大块擦除 

        。

      - 速度：NOR随机读取速度快，NAND串行读写速度快 

        。

      - 性能：带有NAND闪存芯片的SSD性能显著高于传统存储介质（如HDD、CD和磁带） 

        。

      - 可靠性：没有移动部件，比HDD更可靠 

        。

      - 延迟：企业级闪存系统能够实现微秒或毫秒级的低延迟 

        。

    - 缺点

      - 成本：SSD比同等容量的HDD昂贵得多 

        。

      - 容量：目前HDD的容量更大（可达12TB），而最大的SSD容量仅为2TB 

        。

      - 损耗机制：过多的编程/擦除周期会导致存储单元失效，最终破坏捕获电子的氧化层 

        。一旦丢失足够多的比特位，整个驱动器可能无法运行 

        。

      - 数据恢复：SSD单元随机损坏后，数据几乎无法读取和恢复 

        。而损坏的HDD，其盘片上的磁化模式不会被破坏，数据恢复相对容易 

        。

      - 损耗缓解：制造商通常内置控制电路，可检查潜在的故障，确保SSD不会在没有警告的情况下耗尽寿命 

        。

- ## **固态硬盘 (Solid State Drive - SSD)**

  - 基本概念

    - 基于硅基板的闪存介质，因此被称为固态设备 

      。

    - 广泛用于消费电子产品和大型数据存储系统 

      。

  - 形态规格 (Form Factors)

    - 

      2.5英寸SSD

      ：安装在传统硬盘驱动器(HDD)的插槽中，使用SATA或SCSI等标准HDD接口 

      。

    - 

      PCIe插卡式SSD

      ：构建在印刷电路板上的固态卡，使用PCIe等标准I/O接口 

      。

    - 

      模块化SSD

      ：安装在DIMM或SO-DIMM插槽中，使用SATA等标准HDD接口 

      。

    - 

      混合硬盘 (Hybrid Hard Drive)

      ：结合了传统HDD（用于大容量存储）和NAND闪存模块（用于快速I/O） 

      。

  - 全闪存SSD vs. HDD

    - 

      架构差异

      ：全闪存存储系统读写数据无需移动部件 

      ，而HDD依赖于驱动臂在盘片上读写数据 

      。

    - 

      性能优势

      ：SSD在性能上具优势，并可能降低运营成本 

      。

    - 

      普及性

      ：能够插入基于x86的个人电脑和服务器中，增加了其普及度 

      。

    - 成本与应用

      - 个人用户：SSD成本更高，通常只用于更昂贵的个人电脑 

        。

      - 企业用户：在服务器中放置闪存的缺点是需要内部构建硬件系统，并从第三方购买和安装存储管理软件 

        。

- ## **现场可编程门阵列 (FPGA)**

  - 基本概念

    - 一种固态设备，可以被配置以实现不同的功能 

      。

    - 与由程序控制固定电路的微控制器不同，FPGA可以被编程成为不同的数字电路 

      。

    - 可被重新编程以实现完全不同的功能 

      。

  - 应用与优势

    - 

      原型设计

      ：工程师可以用FPGA并编写硬件描述语言(HDL)来控制系统，作为微控制器的替代方案 

      。

    - 

      成本效益

      ：对于大众消费产品，使用FPGA制作原型更便宜，因为其HDL程序可用于设置掩模，以构建生产成本极低的专用集成电路(ASIC)芯片 

      。

  - FPGA 结构

    - 

      I/O块 (IOB)

      ：允许FPGA与外部世界连接 

      。

    - 

      软逻辑块

      ：包含查找表、多路复用器和移位寄存器，可用于构建任何组合或时序电路 

      。

    - 

      硬逻辑块

      ：在本例中由内存块和乘法器组成 

      。实际FPGA中可能还包括通信块、数字信号处理块，甚至整个微处理器 

      。

    - 

      互连资源

      ：开关块(SB)、连接块(CB)和布线通道(Routing Channel)允许软逻辑块内部及不同块之间进行灵活连接 

      。

    - 

      编程机制

      ：HDL程序使用闪存来设置不同部分之间的连接（允许或阻止电流流动） 

      。这些连接可以在需要时被擦除，例如在调试或项目结束后复用FPGA时 

      。

- ## **Verilog (硬件描述语言)**

  - 基本概念

    - 一种硬件描述语言(HDL) 

      。

    - 常用于教学 

      。

  - 与VHDL的对比

    - Verilog比工业界使用的VHDL更简单易学，但能完成几乎所有VHDL能做到的功能 

      。

  - 代码描述方式

    - 

      结构化描述

      ：需要预先有电路图才能编写Verilog程序 

      。例如文档中SR锁存器的代码 

      。

    - **行为化描述**：一种更强大的代码编写方式 。例如文档中D触发器的代码 。
